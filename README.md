## Adding a new game later

1. Make a new folder here (use a short, lowercase, no-spaces name, e.g. `space-maze`).
2. Put the game's `index.html` and other files inside it.
3. Add a card for it in this repo's top-level `index.html`.
4. Commit and push — it goes live at `https://podskarbi.github.io/Games/<folder>/`.

## Updating a game

Edit the files in the game's folder, then commit and push. The website updates by
itself in a minute or two.

## Good to know

- **No accounts, no logins, no ads.** Everything runs in the browser.
- Each game saves its own progress on the device only — nothing is collected or sent anywhere.
